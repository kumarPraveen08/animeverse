# animeverse

> Backend API for anime quotes

## Simple Usage

Rename config.env.env to config.env and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# run in dev mode
npm run dev

# run in prod mode
npm start
```

## API Endpoints

```
# Get quotes
{URL}/api/v1/quotes

# Get quotes
{URL}/api/v1/quotes/random

# quote limit & page
{URL}/api/v1/quotes?page=2&limit=10
```
